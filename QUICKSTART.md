# XMvideo 快速启动指南

## 1. 安装 FFmpeg

XMvideo 依赖 FFmpeg 进行视频处理。首先安装 FFmpeg：

```bash
# 使用 Homebrew 安装
brew install ffmpeg

# 验证安装
ffmpeg -version
```

## 2. 打开项目

```bash
# 进入项目目录
cd XMvideo

# 打开 Xcode 项目
open XMvideo.xcodeproj
```

## 3. 构建并运行

在 Xcode 中：
1. 选择目标设备：My Mac
2. 按 ⌘R 或点击运行按钮
3. 应用会在菜单栏显示一个视频图标

## 4. 使用应用

### 添加视频文件
1. 点击菜单栏的视频图标
2. 拖拽视频文件到界面中
3. 或者拖拽文件夹（会扫描其中的视频）

### 选择压缩配置
在底部选择器中选择：
- **默认**：平衡质量和速度（推荐）
- **快速**：快速处理，质量略低
- **高质量**：高质量输出，处理较慢

### 开始压缩
1. 点击"开始"按钮
2. 查看实时进度
3. 完成后在"历史"标签查看结果

### 高级设置
切换到"设置"标签：
- 删除音频轨道
- 删除源文件
- 使用硬件加速
- 递归扫描文件夹
- 自定义输出目录

## 5. 输出文件

默认情况下，压缩后的文件会保存在与源文件相同的目录下，文件名格式：
```
原文件名_compressed.mp4
```

例如：
- 输入：`video.mov`
- 输出：`video_compressed.mp4`

## 6. 查看历史

切换到"历史"标签可以看到：
- 所有压缩记录
- 总节省空间
- 平均压缩率
- 处理文件数量

## 7. 退出应用

右键点击菜单栏图标，选择"退出 XMvideo"

## 常见问题

### Q: 提示 FFmpeg 未安装
A: 运行 `brew install ffmpeg` 安装 FFmpeg

### Q: 无法拖拽文件
A: 确保应用有文件访问权限

### Q: 压缩速度慢
A: 尝试以下方法：
- 选择"快速"配置
- 启用硬件加速（设置标签）
- 关闭其他占用 CPU 的应用

### Q: 输出文件质量不满意
A: 选择"高质量"配置，或在设置中调整参数

## 技巧

1. **批量处理**：拖拽整个文件夹，启用"递归扫描"
2. **节省空间**：启用"删除源文件"（谨慎使用）
3. **快速压缩**：启用"硬件加速"和选择"快速"配置
4. **移除音频**：如果不需要音频，启用"删除音频轨道"

## 支持的格式

输入格式：MP4, MOV, AVI, MKV, FLV, WMV, M4V
输出格式：MP4 (H.264)
